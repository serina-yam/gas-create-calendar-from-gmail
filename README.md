# gas-create-calender-from-gmail
GMailの情報を取得して予定をGoogle Calendarに登録するGAS。  
該当のGMailはカレンダー登録処理の後に既読にする。

GASの自動実行のタイミングは適宜設定。  
（私は15分おきに設定）
